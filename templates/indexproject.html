PPE Safety Kit Object Detection

Overview

Objective: Detect PPE (Personal Protective Equipment) using YOLO (You Only Look Once) object detection algorithm. The system identifies items such as:

  • Hardhat
  • Mask
  • No-Hardhat
  • No-Mask
  • No-Safety Vest
  • Person
  • Safety Cone
  • Safety Vest
  • Machinery
  • Vehicle

Use Case: Designed for organizations and individuals to ensure workplace safety and reduce hazards effectively.

Dataset

Source: Roboflow's PPE Detection dataset.
Details:

  • Training Images: 2.6k
  • Validation Images: 114
  • Testing Images: 82
  • Image Dimensions: 640x640

This curated dataset ensures high accuracy in detecting PPE equipment across varied environments.
